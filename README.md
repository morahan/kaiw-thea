# Thea Dashboard

Brand strategist dashboard for Workout Flow. Built with React + Vite.

## Features

- **Dashboard** — Key metrics, pending reviews, brand pulse
- **Reviews** — Review queue with quick checklist and verdict actions
- **Voice** — Fish Speech TTS controls and testing
- **History** — Review history with verdict tracking

## Tech Stack

- React 19 + Vite
- Vanilla CSS (no external UI libraries)
- Fish Speech TTS (voice engine)

## Development

```bash
npm install
npm run dev
```

## Deploy

Automatic deploys via Vercel on push to main.

## Brand Standards

- Primary: Teal #0D9488
- Accent: Orange #F97316
- Dark theme with Athenian elegance
- Font: system-ui
